### Projetos em Destaque

* **Automação de Certificações:** Desenvolvimento de solução com Google Apps Script e JavaScript integrando Google Sheets e Slides, reduzindo um trabalho manual de semanas para poucas horas na geração e distribuição de mais de 1.800 certificados acadêmicos.
* **Triagem Inteligente de Currículos (AMEOAM):** Criação de sistema automatizado para o setor de RH, desenvolvido com Python, integração com LLM via API e interface em Streamlit, focado na mitigação de riscos ocupacionais e otimização de recrutamento.
* **Sabor na Medida:** Desenvolvimento de aplicativo mobile utilizando React Native e Firebase, atuando como um marketplace direcionado ao público universitário.
* **Gov-Tech Solutions:** Criação de aplicações web e materiais de revisão responsivos voltados à preparação para concursos públicos em Manaus, com eficácia validada na prática através de aprovações próprias.
